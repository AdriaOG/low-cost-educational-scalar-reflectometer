# Software – Automation and Visualization Scripts  
**Scalar Reflectometer Educational Platform**

This folder contains the two scripts required to perform automated measurements and generate the comparative plots shown in Section IV of the paper.

- `rigol_power_frequency_sweep.py` → Controls the RIGOL DSG821 RF signal generator and the reflectometer microcontroller simultaneously.
- `plot_reflectometer_vs_VNA.m` → MATLAB script that compares the reflectometer measurements against the reference VNA data (AntennaVNA.s1p).

These scripts are fully commented and designed to be easily adapted by students (Stage 8 and Stage 9 of the laboratory sequence).

## 1. Python Script: Automated Power & Frequency Sweep (`rigol_power_frequency_sweep.py`)

**Purpose**  
Performs a 2D sweep (frequency × input power) using the RIGOL DSG821 generator and reads the ADC values (RA0 = forward, RA1 = reverse) from the PIC18F4321 via serial port.  
Saves one CSV file per power level: `measures_0dBm.csv`, `measures_5dBm.csv`, …  

**Key Configurable Variables** (edit at the top of the script)

RIGOL_IP = "192.168.1.100"          # IP of your RIGOL generator
SERIAL_PORT = "COM3"                # Serial port of the reflectometer
START_FREQ_HZ = 50e6                # Start frequency
STOP_FREQ_HZ  = 2.1e9               # Stop frequency
STEP_HZ       = 10e6                # Frequency step → change here
POWERS_DBM    = list(range(0, 21, 5))   # Example: 0, 5, 10, 15, 20 dBm
#POWERS_DBM = list(range(-30, 21, 2))  # Uncomment for finer power sweep
Special Function
transform_csv_to_excel(input_csv, output_xlsx)
→ Converts each measures_XXdBm.csv into Excel format required by the MATLAB script, applies unit conversion (Hz → MHz and raw ADC → volts), and renames columns.
Usage
Bashpython rigol_power_frequency_sweep.py
After finishing, run:
Pythontransform_csv_to_excel("measures_0dBm.csv", "measures_0dBm.xlsx")
# repeat for each power level or automate in a loop
Adaptation
The script uses standard SCPI commands (FREQ, POW, OUTP). It can be easily extended to any other generator of the RIGOL DSG family or any instrument supporting pyvisa/SCPI by changing only the connection string and command syntax.
2. MATLAB Script: Comparison with VNA Reference (plot_reflectometer_vs_VNA.m)
Purpose
Loads the reference S-parameter file from a professional VNA and overlays the reflectometer measurements (for different input power levels) to evaluate dynamic range, sensitivity and agreement with the reference.
Key Configurable Variables
matlabs1pFile = 'AntennaVNA.s1p';           % Reference file from VNA (DUT = antenna)

values_dBm = 0:5:20;                  % Powers to plot → change here (must match Excel files)
% Example: values_dBm = [0 5 10 15 20];
Required files in the same folder

AntennaVNA.s1p          → Touchstone file exported from the VNA (reference |S₁₁|)
measures_0dBm.xlsx, measures_5dBm.xlsx, … (generated by the Python transform function)

Features

Automatic plotting of VNA reference + all selected power levels
Interactive legend (click on a curve to hide/show it)
Pre-configured axes and limits matching the figures in the paper

Usage

Place all Excel files and the .s1p file in the same folder as the script.
Open and run plot_reflectometer_vs_VNA.m in MATLAB.
Modify values_dBm and re-run to explore different power configurations.

Workflow Summary (for students – Stage 8 & 9)

Configure and run rigol_power_frequency_sweep.py → generates CSV files
Convert CSVs to Excel using transform_csv_to_excel(...)
Run plot_reflectometer_vs_VNA.m → obtain publication-quality comparison plots

All scripts are intentionally kept modular so students can experiment with different frequency steps, power ranges, calibration improvements, or even replace the RIGOL generator with another instrument.
